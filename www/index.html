Getting Started

Blitz Scorer is a simple app for keeping score. It was originally designed for the popular game, Dutch Blitz. However, it can used to score any game that scores in rounds such as, 500, Hearts, etc.

To enter a score, tap the cell you wish to change, then adjust the score by tapping a combination of the / buttons at the bottom of the screen.

Change a players name by tapping the name of the player to change.

Start a new round by tapping the button.

Start a new game by tapping the Reset button.

Select the number of players by using the Players menu.

Blitz Scorer is also available on the web, Android, iOS, Windows Phone, FirefoxOS, Chrome App.
